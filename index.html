docker 玩耍记录

Thu, Sep 24, 2015

Docker 1. No more sudo for docker 操作 docker 的时候每次都需要 sudo 来提升权限, 比较麻烦，可以有一个简单的方法来解决。 sudo groupadd docker # 先添加 docker 用户组 sudo gpasswd -a username docker # 把当前使用的账号添加到 docker 组 sudo service docker restart # 重启 docker 服务 2. kafka 命令

Wed, Sep 23, 2015

kafka 常用工具命令 查看topic ./bin/kafka-topic.sh --zookeeper 22288 查看topic 分布 ./bin/kafka-topics.sh --describe --zookeeper localhost:22288 --topic activeDegree 查看topic各个分区的offset ./bin/kafka-run-class.sh kafka.tools.GetOffsetShell --broker-list localhost:9099 --topic activeDegree --time -1 其中 time: -1 表示最新消息的offset, -2 表示最旧的消息的offset, 那么现在的消息条数，还需要计算一下。 (不知道是否有直接计算的命令)
